# Installing OpenHarness for Codex

## Quick Install

```bash
git clone https://github.com/user/openharness.git ~/.codex/openharness
```

Or if you already have the repo locally:

```bash
openharness install codex
```

## After Install

1. Restart Codex
2. Navigate to your project: `cd /path/to/your/project`
3. Initialize: `openharness init`
4. Start working: `openharness start-task "your goal"`
