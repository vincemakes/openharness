Create a new OpenHarness development task.

1. Run `openharness start-task "$ARGUMENTS"` to create the task
2. Read the generated task.md
3. Fill in constraints, success criteria, and assumptions
4. Run `openharness advance` to move to planning
5. Invoke the `openharness-task` skill to continue the lifecycle
